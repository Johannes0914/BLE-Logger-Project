Project Overview

This project focuses on developing a multi-sensor logger and gateway system using Bluetooth Low Energy (BLE). It includes sensor nodes to collect environmental data and a gateway for data aggregation and transmission to Grafana for real-time visualization. The system ensures efficient data logging and monitoring for applications such as agriculture, smart buildings, and industrial equipment monitoring.
